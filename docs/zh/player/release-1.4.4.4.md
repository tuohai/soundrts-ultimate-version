# SoundRTS 1.4.4.4 发行说明

**版本**：1.4.4.4  
**类型**：功能更新 + CrazyMod 内容 + 平衡与修复  
**适用**：单机对战电脑、随机地图、成就/军械库/战前选卡用户；CrazyMod 多分支玩家

---

## 本次更新一览

1.4.4.4 在 1.4.4.3 成就与战前卡牌系统之上，新增 **延迟卡牌**、完善 **结算评分与字母评级**，并加入 **按阵营成就**、**跨阵营元进度**（CrazyMod 等），同时修正战前开局与若干体验问题。

---

## 新功能

### 延迟战前卡牌

战前携带的卡不再只能「开局立刻到账」。现在可以携带 **延迟增援卡**、**延迟科技卡**：

- 选卡进局后，游戏会提示「效果将在若干分钟/秒后生效」。
- 时间到后语音播报「携带卡牌效果生效」，单位在起始位置附近出现，或科技自动研究完成。
- **充能在进局时扣除**，与即时卡相同；到点前请保护好基地，增援不会提前到达。

**如何获得（基础版）**

| 卡牌 | 效果 | 获取方式 |
|------|------|----------|
| 延迟步兵增援卡 | 10 分钟后 3 名步兵 | 成就「援军合约」：胜利并击败中级或高级电脑 |
| 延迟近战武器卡 | 8 分钟后获得近战武器科技 | 成就「击败专家」相关奖励 |

需达到卡牌要求的最低军衔（如中士、少尉）方可在战前栏位选用。

### 结算评分说明

非战役对局结束后，会按多维度播报分数与 **S～E 字母评级**：

- 胜负、采集、效率、存活、建筑、战斗、拆迁七项合计 **800 分**；击败更强电脑另有加分。
- **失败局最高评级为 D**，即使战斗表现很好也无法在本局评 C 及以上（鼓励以胜利为目标）。
- **胜利且花钱很少**时，效率项会按「节约效率」计分并单独播报。
- 详细规则见 [score-and-grades.md](score-and-grades.md)。

### 按阵营成就（CrazyMod 等多分支模组）

- 各阵营拥有 **独立的奖章、军衔、成就与军械库**。
- 主菜单 → **成就**：先选阵营，再查看列表或军械库；从子菜单返回会 **再次选择阵营**。
- 进度保存在 `user/achievements/<模组名>/<阵营名>.json`。
- **战役** 不计入成就、奖章、军衔与结算分数。

### 跨阵营元进度

- 主菜单 → **成就** → **跨阵营进度**：元成就列表与元军械库。
- 在多个分支都有进展后可解锁如 **三分天下**、**五军下士**、**八方巡礼**、**十全霸主**（CrazyMod 示例）。
- 元荣誉为额外称号；元奖章不计入单分支军衔。

### 地图与累计成就（CrazyMod）

- 各地图里程碑 **按阵营分开**；累计击败电脑按分支统计，旧存档难度键名自动迁移。

## 平衡与数据调整

- **少尉**军衔门槛：**200 奖章**（达到后解锁 1 个战前卡牌栏位）。
- **完美生存**成就：要求单位存活维度 ≥90 **且** 建筑保全维度 ≥90（须胜利）。
- **击败初级电脑**重复完成：每次额外 **8 枚**奖章。
- 建筑损失对评分的影响减轻（每座 **5 分**，原为 10）。
- CrazyMod：`rules.txt` / `ai.txt` 多阵营单位与建筑费用、AI 等小幅平衡（虫族建筑仍 0 金）。

---

## 修复与体验

- 部分模组工人「可采集全部资源」时，单位属性不再重复朗读两遍「全部」。
- **战前开局**：无卡牌或无携带栏位时 **不再播报「选择携带卡」**；随机阵营改为提示 **「选择本局阵营」**（不再与选卡混淆）。
- 狩猎动物等 NPC：仅特定玩家击败时才播报并退出，普通地图 NPC 不触发成就向播报。
- 内部测试稳定性改进（模组切换后状态恢复）。

## 升级提示

- 覆盖安装即可；`user/achievements/` 存档 **无需删档**，旧累计击败数据会自动迁移。
- 战役、合作战役、联机：**仍不参与**成就、战前选卡与结算评分播报。
- 若模组设置了 `achievements_enabled 0`，整套成就/卡牌/军衔系统保持关闭。

## 测试建议

1. 主菜单 → 成就 → 军械库，查看延迟卡说明是否朗读「效果将在…后生效」。
2. CrazyMod：选两个不同阵营，确认奖章互不影响；跨阵营进度是否显示。
3. 随机阵营开局：无卡时不提「携带卡」；有延迟卡时约 10 游戏分钟后增援到达。
4. 完成一局非战役对战，听结算分数与字母评级播报。

```bash
python -m pytest soundrts/tests/test_card_loadout.py soundrts/tests/test_score_breakdown.py soundrts/tests/test_meta_progress.py soundrts/tests/test_faction_progress.py -q
```
