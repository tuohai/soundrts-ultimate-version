# 携带物品到达与剧情交付说明（has_brought_item + remove_item）

本说明介绍两个配合使用的战役/地图功能：

- **`has_brought_item`**（触发器**条件**）：玩家单位**携带**指定物品**到达**某方格
- **`remove_item`**（触发器**动作**）：从玩家库存**移除并销毁**物品，用于剧情“上交”

典型场景：玩家把魔法药剂带到祭坛 → 播放过场「守护者早已等候多时，你将药剂交给了她」→ 药剂从背包消失 → 目标完成。

示例地图：The Legend of Raynor 第 18 章（[18.txt](../../../res/single/The Legend of Raynor/18.txt)）。

---

## 1. 与其它物品触发器的区别

| 条件/动作 | 检测/效果 | 物品位置 | 典型用途 |
| --- | --- | --- | --- |
| `has_item` | 玩家持有物品 | 任意单位库存 | 找到/拾取某物 |
| `has_brought_item` | 玩家携带物品到达方格 | 该方格上单位的库存 | 运送到达（无需丢弃） |
| `find` | 方格地面有该物体 | 地上（须先丢弃） | 放到指定地点；语法：**方格在前** `(find c3 mana_potion)` |
| `npc_has_item` | NPC 收到物品 | NPC 库存 / `received_items` | 右键交给 NPC |
| `remove_item` | 从玩家库存删除物品 | 销毁 | 剧情自动上交 |

---

## 2. 条件：`has_brought_item`

```
(has_brought_item <方格> <物品type_name> [数量])
```

- **方格**：如 `c3`、`"3,3"`
- **物品 type_name**：如 `mana_potion`
- **数量**：可选，默认 `1`

判定：在目标方格上，该玩家至少有一个存活单位，其库存中合计持有足够数量的指定物品。

- 空手进入方格 → **不成立**
- 在别的格持有物品、人未到达目标格 → **不成立**
- 携带物品走到目标格 → **成立**（无需丢弃）

示例：

```
trigger player1 (has_brought_item c3 mana_potion) (objective_complete 1)
```

---

## 3. 动作：`remove_item`

```
(remove_item <物品type_name> [方格] [数量])
```

- **未写方格**：从该玩家所有存活单位库存中移除
- **写了方格**（如 `c3`）：只从该方格上的玩家单位库存中移除
- **数量**：可选，默认 `1`

物品会从库存移除并销毁（与使用消耗品类似），可配合 `cut_scene` 表示剧情交付。

---

## 4. 完整示例（第 18 章）

地图要点：

- `c3` 放置中立祭坛建筑 `shrine`
- 玩家在 `b1` 拾取 `mana_potion`，携带走到 `c3`

触发器：

```
trigger player1 (has_brought_item c3 mana_potion)
    (do (cut_scene 7560) (remove_item mana_potion c3) (objective_complete 1))
```

> 须用 ``do`` 串联多个动作。``if`` 只有 if/else 两分支，不能写成
> ``(if (timer 0) (cut_scene ...) (remove_item ...) (objective_complete ...))``，
> 否则条件成立时只会播放过场，后两项不会执行。

流程：

1. 农民拾取魔法药剂
2. 走到 c3 祭坛（药剂仍在背包）
3. `has_brought_item` 成立
4. 播放过场 7560（祭坛守护者等候、收下药剂）
5. `remove_item` 使药剂从背包消失
6. 目标 1 完成

---

## 5. 与「交给 NPC」如何选择

| 方式 | 适用 |
| --- | --- |
| `npc_has_item` + 玩家右键 `give` | 地图上有实体 NPC，物品进入 NPC 库存 |
| `has_brought_item` + `remove_item` | 到达即交付、无实体接收者、或自动剧情上交 |

两种均可配合 `cut_scene` 讲故事。
