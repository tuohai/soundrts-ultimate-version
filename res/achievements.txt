; ===========================================================================

; Achievements (post-game unlocks, phase 1)

; ===========================================================================

;

; Each achievement is a ``def <id>`` block. Mods can append or replace via

; ``clear`` (same layering as ai.txt — see docs/zh/achievement-system.md).

;

; Directives:

;   title <tts_id>           voice title (define id in ui/tts.txt)

;   condition grade S        match settlement grade letter (S/A/B/C/D/E)

;   condition victory        player won the game

;   condition utilization_below <n>   resource use percent strictly below n (victory only)

;   condition defeated_ai <tier> [<tier>...]   defeated any listed AI (OR)
;   condition map <map_id> [<map_id>...]       played on listed map (OR); use with victory + defeated_ai
;   condition defeated_ai_total_at_least <n> <tier>...  cumulative wins vs tier (easy→beginner, aggressive→intermediate)
;   condition defeated_ai_map_at_least <map> <n> <tier>...  cumulative wins on map vs tier

;   condition survival_at_least <n>   unit survival score >= n

;   condition building_defense_at_least <n>

;   once                     unlock at most once ever (default)

;   once_per map             once per map name

;   once_per map_ai          once per map + hardest defeated enemy AI tier

;   reward medal <n>         grant n medals to armory inventory

;   reward card <card_id> [<times>]   grant card charges (see cards.txt)

;   reward title <title_id>  optional honor epithet (only when name differs from achievement)

;   repeat_medal <n>         on repeat completion (same once-key), grant n medals only

;

; All ``condition`` lines in one block must be satisfied (AND).
