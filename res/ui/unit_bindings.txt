; 单位选择界面热键
; 以下默认命令键与全局层相同，可在此单独覆盖（仅单位界面生效）

BACKSPACE: default
SHIFT BACKSPACE: default queue_order
CTRL BACKSPACE: default imperative
CTRL SHIFT BACKSPACE: default imperative queue_order

; 批量选择士兵
a: select_units local SOLDIER
CTRL a: select_units SOLDIER
SHIFT a: select_units local worker SOLDIER
CTRL SHIFT a: select_units worker SOLDIER
ALT a: select_units local worker SOLDIER
CTRL ALT a: select_units worker SOLDIER

; 逐个切换单位
q: select_unit 1 local
SHIFT q: select_unit -1 local

CTRL SHIFT q: select_unit -1

; 命令快捷字母（style.txt 中 order 的 shortcut）

; 过滤器（m-n，选目标时）
m: toggle_side_filter 1

n: toggle_type_filter 1
SHIFT n: toggle_type_filter -1
