主语音与副语音（玩家向）
========================


游戏使用两套可独立配置的语音库：主语音与副语音。可启用或禁用副语音；禁用后由主语音接管全部播报。

**建议：** 尽量把**读屏软件 / 屏幕阅读器**（争渡、NVDA 等）当作主语音。读屏激活时会接管主库职责，你就不必再为「调节主库」占用 ``F9``–``F12`` 等键位。本游戏热键非常密集、几乎饱和，很容易冲突，**能省则省**；副库仍可用 ``Shift+F9``–``F12`` 等键调节战场播报。


----


职责分工
--------


.. list-table::
   :header-rows: 1

   * - 语音库
     - 负责什么
   * - **主语音库**
     - 菜单、玩家操作（选中、移动、模式切换等）；对局外的全部播报；以及对局内**己方经济/生产反馈**（单位/建筑完成、科技研究完成、时代升级完成、资源库存变化、菜单已改变等）
   * - **副语音库**
     - 对局内**战场被动事件**（发现敌人、伤亡、侦察情报、战斗格警报、世界消息等）

启用副语音时：主库与副库可并行。

打断 / 跳过当前播报：

- **左 Alt**：过滤（跳过 / 停止）**主语音库**
- **右 Alt**：过滤（跳过 / 停止）**副语音库**

禁用副语音时：主语音播报一切（与单通道模式相同）；**左 Alt 与右 Alt 均可跳过当前播报**（此时没有副库可过滤）。


----


如何打开设置
------------


1. 主菜单 → **选项** → **语音库设置**
2. 可选择：
   - **启用或禁用副语音**（也可在任意菜单按 **F3**；对局内无效）
   - **主语音库** / **副语音库**：分别调节音量、音调、语速、音色、声卡
   - **打开语音安装文件夹**：打开 ``user/voices``，便于安装音库


----


调节参数与热键
--------------


在「主/副语音库」编辑器中：

- 上下光标：选择参数（音量 / 音调 / 语速 / 语音 / 声卡）
- 左右光标：调整当前参数
- 回车或 Esc：返回

局内（及菜单）快捷键：

- **F9–F12**：调节主语音库（声卡 / 参数类型 / 减小 / 增大）
- **Shift+F9–F12**：调节副语音库（任意一侧 Shift）
- **右 Shift+C**：复制副库最近播报；**右 Shift+B**：追加副库最近播报到剪贴板
- **左 Shift+C / 左 Shift+B**（复制/追加主库）：默认在 ``res/ui/global_bindings.txt`` 里**已注释**，以减少热键冲突；若要用，去掉对应行前的 ``;`` 即可启用
- **菜单内 F3**：启用或禁用副语音（对局内不绑定此功能）


----


如何添加语音库
--------------


打开「打开语音安装文件夹」后，目录一般为 ``user/voices``。

SAPI 系统语音
~~~~~~~~~~~~~

1. 先在 Windows 中安装并注册 SAPI5 语音（控制面板 / 设置 → 语音）。
2. 回到游戏，在主或副语音库里用「语音」参数循环选择已安装的音色。
3. 部分仅 32 位的音库可由游戏通过 ``tools/sapi32`` 调用。

语音包文件夹（友好名称）
~~~~~~~~~~~~~~~~~~~~~~~~

在 ``user/voices`` 下新建子文件夹，放入 ``voice.ini``，例如::

    [voice]
    title=我的中文名
    sapi=Microsoft Huihui Desktop
    rate=0

- ``title``：菜单里显示的名称
- ``sapi``：必须能匹配系统中已注册的 SAPI 语音名（可子串匹配）
- ``rate``：可选，约 -10～10

此类包**不会**安装 TTS 引擎，只是给已有 SAPI 语音起别名。

Nuance / 苹果音库（可选）
~~~~~~~~~~~~~~~~~~~~~~~~

若已将 Nuance 数据包放到 ``user/voices/nuance``，可在语音列表中选到对应音色。具体导入方式见该目录说明或发行包附带工具。

主 / 副语音库**都可以**只选 SAPI，不必使用 Nuance。Nuance 需要发行包中的 32 位 Java 助手（``tools/nuance_ve``）；SAPI 路径不依赖 Java。


----


对局内方向播报（立体声定位）
----------------------------


对局内部分**带格子位置**的被动播报（例如发现敌人、伤亡、侦察情报、战斗格警报）会按该格相对**当前视角格子**的方位，从左 / 右（及前后衰减）播出，定位算法与小地图警报音效同源。

建议戴耳机聆听。默认俯视朝北时：东侧格子偏右，西侧偏左。

**音量下限：** 文字播报的距离衰减不会无限变轻——远端格子大约保持在「相邻一格」的音量（可略弱一点），便于操作中听清；左右与后方衰减仍在。小地图警报蜂鸣仍按完整距离衰减。

**播报中途换格会跟新：** 例如你在 b1 时听到左边 a1 有敌人，尚未说完就切到 a1，声道会收到中间（正前方），无需等到下一条消息。

说明：

- 方向由副语音库（启用时）或主语音库（副库关闭时）的文字播报承担；与音色是 SAPI 还是 Nuance 无关。
- 菜单、选中单位等操作播报仍为居中，不做方向定位。
- 读屏接管主语音时，主库方向定位不可用；副库仍可按上方式定位。


----


读屏软件
--------


若检测到争渡、NVDA 等专用读屏，**主语音职责**可由读屏接管，避免与主库抢播；副语音（启用时）仍使用游戏内配置的副库。
