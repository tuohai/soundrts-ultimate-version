主语音与副语音（玩家向）
========================


游戏使用两套可独立配置的语音库：主语音与副语音。可启用或禁用副语音；禁用后由主语音接管全部播报。


----


职责分工
--------


.. list-table::
   :header-rows: 1

   * - 语音库
     - 负责什么
   * - **主语音库**
     - 菜单、玩家操作（选中、移动、模式切换等）、对局外的全部播报
   * - **副语音库**
     - 仅在对局内播报被动事件（伤亡、发现、世界消息等）

启用副语音时：对局内操作走主语音，被动事件走副语音，两者可并行；只有 **Alt** 会打断副语音。

禁用副语音时：主语音播报一切（与单通道模式相同），玩家操作会打断正在进行的被动播报。


----


如何打开设置
------------


1. 主菜单 → **选项** → **语音库设置**
2. 可选择：
   - **启用或禁用副语音**（也可在任意菜单按 **F3**；对局内无效）
   - **主语音库** / **副语音库**：分别调节音量、音调、语速、音色、声卡
   - **打开语音安装文件夹**：打开 ``user/voices``，便于安装音库


----


调节参数与热键
--------------


在「主/副语音库」编辑器中：

- 上下光标：选择参数（音量 / 音调 / 语速 / 语音 / 声卡）
- 左右光标：调整当前参数
- 回车或 Esc：返回

局内（及菜单）快捷键：

- **F9–F12**：调节主语音库（声卡 / 参数类型 / 减小 / 增大）
- **Shift+F9–F12**：调节副语音库
- **左 Shift+C**：复制主库最近播报；**右 Shift+C**：复制副库最近播报
- **Shift+A**：追加到剪贴板
- **菜单内 F3**：启用或禁用副语音（对局内不绑定此功能）


----


如何添加语音库
--------------


打开「打开语音安装文件夹」后，目录一般为 ``user/voices``。

SAPI 系统语音
~~~~~~~~~~~~~

1. 先在 Windows 中安装并注册 SAPI5 语音（控制面板 / 设置 → 语音）。
2. 回到游戏，在主或副语音库里用「语音」参数循环选择已安装的音色。
3. 部分仅 32 位的音库可由游戏通过 ``tools/sapi32`` 调用。

语音包文件夹（友好名称）
~~~~~~~~~~~~~~~~~~~~~~~~

在 ``user/voices`` 下新建子文件夹，放入 ``voice.ini``，例如::

    [voice]
    title=我的中文名
    sapi=Microsoft Huihui Desktop
    rate=0

- ``title``：菜单里显示的名称
- ``sapi``：必须能匹配系统中已注册的 SAPI 语音名（可子串匹配）
- ``rate``：可选，约 -10～10

此类包**不会**安装 TTS 引擎，只是给已有 SAPI 语音起别名。

Nuance / 苹果音库（可选）
~~~~~~~~~~~~~~~~~~~~~~~~

若已将 Nuance 数据包放到 ``user/voices/nuance``，可在语音列表中选到对应音色。具体导入方式见该目录说明或发行包附带工具。


----


读屏软件
--------


若检测到争渡、NVDA 等专用读屏，**主语音职责**可由读屏接管，避免与主库抢播；副语音（启用时）仍使用游戏内配置的副库。
