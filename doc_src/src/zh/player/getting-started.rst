SoundRTS 玩家入门指南
=====================



用耳朵指挥千军万马——这是一份由浅入深的阅读路线。你可以按顺序通读，也可以跳到对应章节。


----


你即将进入的世界
----------------


SoundRTS 是一款音频即时战略游戏：没有画面，全靠声音。你将在格子地图上探索、采集、建造、练兵、作战——玩法灵感来自《魔兽争霸》与《星际争霸》，并专为视障玩家设计，同样适合喜欢「闭眼指挥」的玩家。

两种视角，随时切换：


.. list-table::
   :header-rows: 1

   * - 模式
     - 怎么进
     - 适合做什么
   * - 地图模式（默认）
     - 启动即在此
     - 宏观调度：选单位、下命令、看资源
   * - 第一人称模式
     - Alt+Space 或 Ctrl+Space
     - 微观操作：亲自走位、放技能



大多数对局以地图模式为主；探索密林、精确走位时可切第一人称。

第一层：十分钟上手
------------------



目标：能选中农民、采金矿、盖一座农场。

建议起点
~~~~~~~~


主菜单 → 单人游戏 → 选第一张教程地图（或任意小图对战电脑）。若使用 Jaws 等读屏软件，有时需先关闭读屏，键盘才不被占用。

五个必会动作
~~~~~~~~~~~~



.. list-table::
   :header-rows: 1

   * - 你想做什么
     - 按键
     - 说明
   * - 选中下一个己方单位
     - Q
     - 反复按直到听到「农民 1，等待命令」
   * - 选中下一座建筑
     - W
     - 例如市政厅
   * - 在列表里选命令
     - A / Shift+A
     - 下/上一个命令
   * - 在列表里选目标
     - Tab / Shift+Tab
     - 金矿、森林、空地等
   * - 确认命令
     - Enter
     - 农民开始工作



更快的方式： 先 Tab 选中金矿，再按 Backspace——系统会执行该单位的默认命令（农民即「开采金矿」）。

资源三键
~~~~~~~~



.. list-table::
   :header-rows: 1

   * - 按键
     - 资源
   * - Z
     - 黄金（resource1）
   * - X
     - 木材（resource2）
   * - C
     - 食物 / 人口



在地图上「飞行」
~~~~~~~~~~~~~~~~


- 方向键：在格子间移动光标（有路会听到脚步声，无路会撞墙）
- Page Up / Page Down：跳到下一个「有内容」的格子，跳过空格子
- Space：跟随当前选中的单位所在格

本章小结
~~~~~~~~


.. code-block:: text

   选单位(Q) → 选命令(A) → 选目标(Tab) → 确认(Enter)
                       或：选目标(Tab) → 默认命令(Backspace)


更完整的按键表见官方 `游戏手册 <../../../player/manual.rst>`_ 第 3 节；游戏内按 F10 可进菜单与帮助。

第二层：从生存到扩张
--------------------



目标：理解 RTS 核心循环，能打赢简单电脑。

经济循环
~~~~~~~~


1. 农民采集黄金与木材
2. 农场提供食物上限——没有足够食物就训不了兵
3. 建筑训练单位、研发科技
4. 科技升级对该玩家所有单位生效（含已有的）

小技巧：

- 给市政厅设集结点（选中市政厅 → Tab 到金矿 → Backspace），新农民会自动去采矿
- 按 D 控制所有农民，批量改派任务
- 资源点采空后，用 Shift+Backspace 给农民排队多个采集目标

军事基础
~~~~~~~~



.. list-table::
   :header-rows: 1

   * - 概念
     - 怎么做
   * - 编组
     - 选中单位后 Shift+6~9 存入编组，6~9 调出
   * - 巡逻
     - 命令「patrol」——扩大警戒范围又保持集中
   * - 防御模式
     - 遇强敌会逃跑而非送死——适合侦察
   * - 强制攻击
     - Ctrl+Backspace——无视沿途敌人，直奔目标
   * - 封锁路口
     - 单位 block 命令，或在路上造墙/建筑



查看战况
~~~~~~~~


- V：当前单位生命值
- M / N：切换 Tab 筛选（友军/敌军、建筑/单位）
- F5 / F6：历史消息上一条/下一条

缩放模式（精细走位）
~~~~~~~~~~~~~~~~~~~~


F8 进入缩放：一个大格子拆成 3×3（可调至 15×15）子格，适合精确布阵或观战。  
Ctrl+Shift+F8 提高精度，Shift+F8 降低。

随机地图
~~~~~~~~


主菜单或联机房间可选 随机地图：设定尺寸、资源、野怪等，生成后播报种子与分享码，可复现同一布局。  
详情 → `随机地图功能说明 <随机地图功能说明.htm>`_

存档与继续
~~~~~~~~~~


- 每模组最多 10 个存档槽；取消对局会留记忆点
- 主菜单 继续未完成的游戏
- 联机可观战（服务器菜单 spectate）

~~~~~~~~~~~~~~~~~~~~

第四层：联机与社交
------------------


快速联机
~~~~~~~~


主菜单 → 多人游戏 → 选服务器 → 创建或加入房间 → 选地图与速度 → 开始。

- F7：聊天（大厅、房间、对局中）
- 团队须在开局前创建；盟友共享视野与胜利
- 动态结盟：F12 选候选，F4 发请求，Ctrl+F4 接受（若地图允许）

条约模式
~~~~~~~~


开战前可选和平时长（最长 20 分钟），适合慢热局。

服务器与端口
~~~~~~~~~~~~


默认 TCP 2500。若外网朋友连不上，需路由器端口转发或 UPnP。  
独立服务器与客户端连接方式略有不同——见 `游戏手册 <../../../player/manual.rst>`_ 第 4.4 节。

启用模组
~~~~~~~~


在 ``user`` 文件夹的 SoundRTS.ini 中设置，例如：

.. code-block:: ini

   mods = soundpack,starcraft


或用命令行：``soundrts.exe --mods=soundpack,orc``  
音效包 mod 可在选项 → 音效包中勾选。

遇到问题？
----------



.. list-table::
   :header-rows: 1

   * - 情况
     - 建议
   * - 语音被下一条打断
     - 增大 ``SoundRTS.ini`` 里 ``wait_delay_per_character``
   * - 按键不对
     - 选项 → 按键映射，或查看 [分层热键方案说明.md](分层热键方案说明.htm)
   * - 想改语言
     - ``cfg/language.txt`` 写入 ``zh``、``en`` 等
   * - 打开用户文件夹
     - 选项菜单中有入口
   * - 单人调试用
     - 反引号控制台：``add_units a1 10 archer``、``victory``
